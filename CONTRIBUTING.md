# Contributing to VidRAG

## Setup
Clone the repository, then run `uv sync` to set up the virtual environment and install dependencies.

## Code style
- Type hints are required for all function signatures.
- Docstrings are required for all public functions and classes.
- Use `black` and `isort` for formatting.

## Testing
Run `pytest` to execute the test suite (coming soon).  
Please open an issue before making large changes to discuss the proposed modifications.
